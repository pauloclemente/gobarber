## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/pauloclemente/gobarber

# Access
$ cd gobarber

# Install dependencies
$ yarn

$ yarn start

# The server will initialize in the <http://localhost:3000>
```
